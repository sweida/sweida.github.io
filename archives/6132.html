vscode 前端最佳配置

工具 vscode 2018/07/28


vscode前端开发的利器，安装上一些好用的插件和修改个人配置文件，更是所向披靡，开发起来快别人几条街。
下面详细解释vscode的个人配置
点击左下角的设置按钮 > 设置 > 右边栏的用户设置，修改user setting文件

另外介绍几款超强大的插件

Settings Sync
同步vscode的插件和配置，更换电脑时就可以用之前保存好的插件和配置了

live server
开启一个服务器，代码更新页面就会自动刷新了

bookmarks
代码标签，代码标记后会保存起来，查找代码就很方便了

path Intelliense
路径提示，妈妈再也不用担心我路径写错迷路了

Bracket Pair Colorizer
括号颜色标记，很好的区分各个代码块
