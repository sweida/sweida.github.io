markdown代码块支持高亮显示的语言

工具 markdown 2018/06/20

markdown里对代码块的引用语法是三个撇[ ``` ],在其后可增加代码名称,比如java,js等标记该部分代码的类型.之后在页面展现的时候就可以高亮显示关键字了.

比如下面这段代码:

```javascript
var xiaoming = {
  name: ‘小明’,
  birth: 1990,
  age: function () {
    var y = new Date().getFullYear();
    return y - this.birth;
  }
};
```
